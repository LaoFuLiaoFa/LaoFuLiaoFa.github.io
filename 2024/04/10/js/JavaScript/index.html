立即执行函数

  • 概念：
    • 专业名字：Immediately-Invoked Function Expression（IIFE立即调用函数表达式）
    • 表达的含义是一个函数定义完后被立即执行；
      • 第一部分是定义一个匿名函数，这个函数有自己独立的作用域；
      • 第二部分是后面的（），表示这个函数被执行了。

  • 应用场景

    • 防止全局变量的命名冲突；
    • 会创建一个独立的执行上下文环境，可以避免外界访问或修改内部的变量，也避免了对内部变量的修改；

  • 立即执行函数必须是一个表达式（整体），不能是函数声明。

面向对象

对象类型

  • 对象类型是一种存储键值对(key-value)的更复杂的数据类型。
    • key：是字符串，也叫做属性名property name。但是在定义对象的属性名时，大部分情况下引号都是可以省略的。
    • value：可以是任意类型，包括基本数据类型、函数类型、对象类型等。

对象的基本使用

术语：函数和方法

  • 函数(function)：如果在JavaScript代码中通过function默认定义一个结构，称之为函数；
  • 方法(method)：如果将一个函数放到对象中，作为对象的一个属性，那么将这个函数称之为方法。

对象的创建方式

  1. 对象字面量( Object Literal)；
  1. new Object( ) + 动态添加属性；
  1. new 其他类( )；
  • 属性之间是以逗号(comma)分割的；

栈内存和堆内存

  • 程序是需要加载到内存中来执行的，可以将内存划分为两个区域：栈内存和堆内存。
    • 原始类型占据的空间是在栈内存中分配的；
    • 对象类型占据的空间是在堆内存中分配的。
